第 54 章
LBO
ローカルブレイクアウト（以下  LBO）は、インターネットアクセスの一部を、各拠点に設置したルーターから直接
行う機能です。
ヤマハルーターは、 LBO の対象となるアプリケーションが使用する  IP アドレスや  FQDN が記載された通信先リス
トを外部から取得し、そのリストをもとにフィルター型ルーティングを用いて  LBO を実現します。
通信先リストはヤマハの  Web サーバー上で提供されており、ヤマハが定期的にメンテナンスを行うため、最新の  IP
アドレスや  FQDN が反映されています。また、ヤマハルーターは定期的にこの通信先リスト（以下オフィシャルリ
スト）を Webサーバーからチェックし、新しいバージョンのリストが公開されていた場合は、自動的に使用中のリ
ストを更新します。ルーターの管理者による通信先リストやルーター設定の定期的なメンテナンスは不要です。
本機能に関する技術情報は以下に示す  URL で公開しています。
https://www.rtpro.yamaha.co.jp/RT/docs/lbo/

54.4 通信先リストの取得の手動実行
[書式 ]
lbo list get go [no-confirm] [force]
[設定値及び初期値 ]
• no-confirm : 新しいバージョンの通信先リストが存在するときに、 通信先リストの更新を行うか否かを確認するメ
ッセージを表示しない
• [初期値 ] : -
• force : 通信先リストのバージョンチェックを行わず、強制的に通信先リストを取得する  また  RTFS に保存された
キャッシュファイルも強制的に更新する
• [初期値 ] : -
[説明 ]
LBO で使用する通信先リストの取得を手動で実行する。
[ノート ]
通信先リストの更新チェックは自動で行われるため、通常は本コマンドを実行する必要はない。
自動での取得に失敗したときなどに、直ちにリトライをしたい場合に使用する。
[適用モデル ]
RTX840

[書式 ]

[説明 ]

•動作状態
表示 説明
無効 LBO が無効になっている。
正常動作中 LBO が正常に動作している。
IPアドレス  / FQDN リストのダウンロード中 通信先リストを取得している。
IPアドレス  / FQDN リストのダウンロード失敗  (リト
ライ待ち )通信先リストの取得に失敗し、再取得中。
IPアドレス  / FQDN リストのダウンロード更新中 通信先リストを更新している。
異常停止中 異常が発生し、 LBO の動作を停止している。
•通信先リストの情報
表示 説明
バージョン番号 通信先リストの中に記述されたバージョン番号
最終取得日時 通信先リストを取得した最終日時
最終更新日時 通信先リストの更新処理を実行した最終日時
最終更新結果更新処理を実行した結果
•最新版に更新  … 最新版の通信先リストに更新した
•更新スキップ  … 最新版の通信先リストを使用中の
ため、更新をスキップした
•更新失敗  … 最新版の通信先リストの取得に失敗し
た

• YES … 利用中
• NO … 利用していない

•application

• [初期値 ] : -
[説明 ]

    PP[1] 425403 オクテット
