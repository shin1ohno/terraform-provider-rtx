55.1 ダッシュボードのデータを蓄積するか否かの設定
[書式 ]
dashboard  accumulate  type sw
no dashboard  accumulate  type [sw]
[設定値及び初期値 ]
•type : ガジェットの種別
• [設定値 ] :
設定値 説明
traffic トラフィック情報
nat NAT セッション数
• [初期値 ] : -
•sw
• [設定値 ] :
設定値 説明
on データを蓄積する
off データを蓄積しない
• [初期値 ] : off
[説明 ]
Web GUI のダッシュボードのガジェットで使用するデータを蓄積するか否かを設定する。なお、ダッシュボードで
type で示されるガジェットを追加すると自動的に本コマンドの設定が  on となる。
[ノート ]
RTX1210 は Rev.14.01.34 以降で使用不可。
RTX830 は Rev.15.02.10 以降で使用不可。
代わりに  statistics  コマンドを使用することができる。
